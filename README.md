# Social-Media-Platform
A UI/UX design of a social media mobile app created in Figma.

## Project Description

This project is a complete UI/UX design of a social media mobile application created using Figma. It includes both low-fidelity (Lo-Fi) wireframes and high-fidelity (Hi-Fi) designs.

The low-fidelity wireframes focus on the basic structure, layout, and user flow of the application, helping to plan the overall functionality and navigation. The high-fidelity designs represent the final polished interface with proper colors, typography, images, and realistic UI components.

The app includes key features such as user login and signup, home feed, profile section, post creation, and smooth navigation. This project demonstrates the full design process from initial concept to final user interface.

## Features

- User Login & Signup  
- Home Feed with posts  
- Profile Screen  
- Create Post functionality  
- Smooth Navigation bar  
- Clean and modern UI design
